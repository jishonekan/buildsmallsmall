# Build Small Small

This is the landing page and auth flow for the Build Small Small platform.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Next.js App Router pages (Home, Login, Register)
- `src/components/landing`: Landing page specific components
- `src/components/ui`: Reusable UI components (Button, Container)
- `src/lib`: Utilities

## Pages

- `/`: Main Landing Page
- `/login`: Login Page
- `/register`: Registration Page

## Tech Stack

- Next.js 14
- Tailwind CSS
- Framer Motion
- Lucide React
